# NewStage V2 (UX-first) — Demo

**NewStage** — прототип сервиса для школьников **9/11 класса**, который помогает **понятно и быстро** выбрать **колледж или вуз Москвы** по ключевым критериям.  
Эта версия — **V2 (UX-first)**: меньше перегруза, один главный сценарий “подбор → 3 варианта → сравнение”.

> ⚠️ Данные и AI-оценки в проекте **демонстрационные** (заглушки). Логика и “умное сравнение” будут подключены позже.

---

## 🚀 Live demo  
`lilcupsup.github.io/newstage-frontend/`

---

## 🎯 Что улучшено в V2
- **Упрощённый интерфейс**: меньше визуальных эффектов и “шума”
- **Один главный сценарий**: пользователь сразу понимает, что делать
- **Мобильный UX**: есть меню-бургер **без JS** (через `<details>`)
- **Меньше выбора = проще решение**: в результатах **3 карточки**, а не 6+
- **Доступность**: `:focus-visible`, контраст, семантика

---

## 🧱 Стек
- **HTML5** (семантическая разметка)
- **CSS3** (dark theme, responsive, a11y)
- **Без JS** (структура готова к подключению логики)

---

## 🗂️ Структура страницы (V2)
- Sticky header + навигация (и burger на мобилке)
- Hero (коротко и ясно)
- Подбор (основные поля + “Доп. параметры” через `<details>`)
- Результаты (3 карточки)
- Сравнение (компактная таблица + короткий AI-вывод)
- FAQ (через `<details>/<summary>`)
- Контакты + Footer

---

## ▶️ Как запустить локально
Вариант 1:
- Просто открой `index.html` в браузере

Вариант 2 (рекомендуется):
- VS Code → расширение **Live Server**
- ПКМ по `index.html` → **Open with Live Server**
