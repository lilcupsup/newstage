> Помощник для выбора колледжа или университета в Москве для учеников 9 и 11 классов

---

## 🌐 Демо

👉 **https://lilcupsup.github.io/newstage/**

---

## 🚀 О проекте

**NewStage** — это веб-сервис, который помогает школьникам выбрать учебное заведение:

- 🔍 найти подходящие вузы  
- ⚖️ сравнить варианты между собой  
- 🤖 получить рекомендацию на основе своих данных  

---

## 🧩 Функционал

### 🔍 Поиск вузов
- Фильтрация по направлению  
- Ограничение по стоимости обучения  
- Учёт баллов ЕГЭ  

### ⚖️ Сравнение
- Добавление вузов в список сравнения  
- Таблица с ключевыми параметрами:
  - Рейтинг  
  - Стоимость  
  - Плюсы и минусы  

### 🤖 ИИ-рекомендации
- Ввод данных:
  - Интересы  
  - Бюджет  
  - Баллы ЕГЭ  
- Генерация рекомендации (имитация ИИ)
